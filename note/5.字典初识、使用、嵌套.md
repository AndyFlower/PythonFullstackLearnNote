## 字典

### 字典的初识

#### why

- 列表可以存储大量的数据，但是数据之间的关联性不强
- 列表的查询速度很慢

#### what

容器型的数据类型

#### how

- 数据类型的分类（可变不可变）

  - 可变（不可哈希）：list  dict set

    ```python
    # 可变
    li = [1,2,3]
    li.append(4)
    print(li) # [1, 2, 3, 4]
    ```

    

  - 不可变（可哈希）：str bool int 

    ```python
    # 不可变
    s ='san'
    s2 = s.upper();
    print(s) #san
    
    ```

    

- 字典：{}括起来，以键值对的方式存在

  ```python
  dict = {'sang':{'name':'sang','age':18},
   '语言':{'python','java','c++'}
   }
  ```

- 键必须是不可变的数据类型 int str 

- 值可以是任意数据类型，对象

- 字典3.5版本之前是无序的

- 3.6会安装初次建立字典的顺序排列

- 3.7 以后是有序的

- 字典的缺点：以空间换时间

- 字典的优点：字典的查询速度非常快
